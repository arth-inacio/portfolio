Latest Projects

Portal da Transparência

O Portal da Transparência é um projeto em Python focado na automação de coleta de dados financeiros públicos através de web scraping. Ele utiliza bibliotecas como Playwright para navegar em sites e extrair informações, e Pandas para manipulação e exportação dos dados em formato CSV. Seu objetivo é facilitar a análise de despesas públicas, sendo útil para fins educacionais e de transparência governamental.

Sistema de Vendas TobaccoRealm

O projeto Sistema de Vendas é uma aplicação desenvolvida em Python com interface gráfica usando Tkinter e armazenamento de dados com SQLite. Ele visa oferecer funcionalidades para gestão de vendas e usuários, apresentando uma interface intuitiva para login, controle de usuários e operações de venda.

About Me

Software Developer

Profissional com sólida experiência em desenvolvimento de software e suporte técnico em TI, especializado em linguagens como Python, JavaScript e PHP. Possui conhecimento em programação assíncrona com foco em web scraping e automação, utilizando ferramentas/frameworks como AsyncPlaywright, Httpx e GitHub. Atualmente, trabalha como Desenvolvedor de Software na Scod Brasil, desenvolvendo aplicações e automações para clientes. Suas principais responsabilidades incluem: - Desenvolvimento de spiders para extração de dados. - Correção e refatoração de código. - Treinamento de novos colaboradores em técnicas de web scraping. Formação Estudante de Análise e Desenvolvimento de Sistemas na Universidade Unopar, complementando sua prática com uma sólida base teórica.
